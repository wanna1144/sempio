# sempio renewal
---------------------------
기여도 100%

```````
 리뉴얼 사이트로 샘표를 선택한 이유
 샘표 사이트의 호버 효과가 타 사이트에서 보지 못했던 효과였고, 사이트 내에 텍스트 슬라이드 등 다양한 효과를 보고 선택하게 되었습니다.
```````
-----------------------------

수정사항
> 1. 메인페이지의 동영상 화질이 너무 깨져 슬라이드로 변경
> 2. 모바일 사이즈 네비게이션에서 서브 메뉴가 있는 메뉴 클릭 시 다른 페이지로 넘어가는 메뉴와 차이가 없어 차이를 알 수 있게 수정
